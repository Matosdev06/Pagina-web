Função da tag (h1 ao h6)

(Voltar)

Bom! a função dessas tags são as mesmas, pois ambas tem o objetivo de criar títulos em páginas web, porem o nível de importancia entre elas é diferente onde a h1 tem o nível mais alto do que as demais h, e a h6 tem o nível de importancia mais baixo.

Função das tags (mark, small e strong)

(Voltar)

Tag mark - Essa tag tem a função de destacar algum trecho de um texto, dando assim mais relevância para o trecho destacado por essa tag.

Ex.: Esta tag é de muita impotância para deixar a citação do seu texto em destaque.

Tag small - Essa tag tem a função de deixar a fonte de um texto menor.

Ex.: Com essa tag a fonte do seu texto ficará menor.

Tag strong - Bom! a tag strong tem a função de deixar as palavras do texto em negrito.

Ex.: Esta frase está em negrito devido a função da tag (strong).

Atribuição das tags (p, i, u, a)

(Voltar)

Tag p - A função dessa tag, será nada mais que dar ínicio e fim ao paragrafo de um texto, ou seja a mesma permite a inserção de parágrafos no conteúdo HTML.

Tag i - Tem a função de deixar as palavras em itálico. Ex.: está tag deixa essa frase em itálico.

Tag u - Tem a função de sublinhar o marco da frase de um texto. Ex.: Essa frase esta sublinhada devido a tag u.

Tag a - A tag a, tem a função de atribuir links no seu texto, obs... para declarar essa tag, devemos incluir o atributo href. Ex.: linkando essa tag.

Função das tags (li e hr)

(Voltar)

Tag li - é usada para representar um item que faz parte de uma lista, este item deve estar contido dentro de um elemento pai, que são as conhecidas como listas ordenadas (ol) e não ordenadas (ul).

Tag hr - representa uma quebra temática entre elementos de um parágrafos, é como uma linha orizontal.

Função das tags (sub e sup)

(Voltar)

Ambas as tags tem a mesma função de escrever textos com letras menores e fazem o alinhamento dos textos, sendo o alinhamento para cima (sup) e o alinhamento para baixo (sub).

Tag sub - é utilizado escrever um texto com letras menores e com o alinhamento abaixo das que do resto do conteúdo, Ex.: É utilizada para escrver um texto com letras menores, com o alinhamento abaixo do resto do conteúdo.

Tag sup - é utilizado escrever um texto com letras menores e com o alinhamento acima das que do resto do conteúdo, Ex.: É utilizado para escrever texto com letras menores e alinhamemto acima do resto do conteúdo.

Função da tag (del)

(Voltar)

Tag del - é usada para identificar o texto que foi excluído de um documento, mas retido para mostrar o histórico de modificações feitas no documento.
